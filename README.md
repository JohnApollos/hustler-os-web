# Hustler OS Landing Page

The official landing page for **Hustler OS**, the zero-effort financial agent for Kenyan entrepreneurs.

## Overview

This is a high-converting, single-page website designed to showcase the features of the Hustler OS mobile app and drive downloads. It features a "Dark Mode / Cyberpunk Fintech" aesthetic with a focus on mobile responsiveness and performance.

## Features

-   **Hero Section**: Compelling value proposition with a "pulse" call-to-action.
-   **Features Grid**: Highlights key app capabilities (M-Pesa Auto-Agent, Receipt Scanner, Smart Invoicing).
-   **Privacy Focus**: Emphasizes data security and local processing.
-   **SME Tools**: Showcases business-centric features.
-   **SEO Optimized**: Includes Open Graph tags, Twitter Cards, JSON-LD structured data, and sitemap.

## Tech Stack

-   **HTML5**: Semantic structure.
-   **Tailwind CSS**: Utility-first styling (via CDN, no build step required).
-   **Font Awesome**: Icons.
-   **Google Fonts**: Roboto (Body) and Montserrat (Headers).

## Setup & Usage

Since this project uses CDN links for dependencies, there is no build process.

1.  Clone the repository:
    ```bash
    git clone https://github.com/JohnApollos/hustler-os-web.git
    ```
2.  Open `index.html` in any modern web browser.

## Deployment

This is a static site and can be deployed to any static hosting provider (GitHub Pages, Vercel, Netlify) by simply serving the root directory.
